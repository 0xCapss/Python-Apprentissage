# Session 02 - Le chiffre de Cesar

## Objectif du jour
Ecrire le premier algorithme de chiffrement en appliquant la methode REGT.
Combiner chaines, arithmetique et modulo.

## Acquis
- Le chiffre de Cesar : decalage de lettres dans l'alphabet avec bouclage.
- Formule cle : `nouvelle_position = (position + decalage) % 26`
- `ord()` et `chr()` : passer d'une lettre a un nombre et inversement.
- `.upper()` : toujours normaliser l'entree avant de calculer.
- En Python, `x % 26` est toujours >= 0 : le dechiffrement par soustraction fonctionne sans condition.
- Dechiffrer = chiffrer avec `26 - decalage` : une seule fonction suffit.

## Cas limites traites
- Minuscules : `.upper()` applique sur l'entree (pas la sortie).
- Bouclage : Z + 3 -> (25 + 3) % 26 = 2 -> C. Valide.
- Espace : detecte avec `lettre == " "` et conserve tel quel.
- Dechiffrement negatif : -3 % 26 = 23 en Python (pas de condition speciale necessaire).
