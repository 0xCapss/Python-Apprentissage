# Session 03 - Tuples et dictionnaires

## Objectif du jour
Decouvrir deux nouvelles structures de donnees : les tuples et les dictionnaires.
Les combiner dans un exercice pratique.

## Acquis

### Tuples
- Sequence immuable : acces par indice comme une liste, mais modification impossible.
- Cas d'usage : donnees constantes par nature (jours, mois, coordonnees GPS).
- Piege : `("valeur")` est une string. `("valeur",)` est un tuple (la virgule fait le tuple).
- Pour construire un tuple progressivement : accumuler dans une liste, puis `tuple(liste)`.

### Dictionnaires
- Structure cle:valeur, acces uniquement par cle (pas par indice).
- Ajouter / modifier : `dict["cle"] = valeur`
- Supprimer : `dict.pop("cle")` retourne la valeur supprimee.
- Parcourir : `.items()` (cle + valeur), `.keys()`, `.values()`.
