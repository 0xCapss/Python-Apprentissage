# Python-Apprentissage - Vault Obsidian

Carnet d'apprentissage Python et Algorithmie, en route vers l'ecole cyber 2600.

## Comment ouvrir

Ce depot est un vault Obsidian.

1. Installe Obsidian (gratuit) : https://obsidian.md
2. Ouvrir un dossier comme coffre, choisis ce dossier (Python-Apprentissage).
3. Tu navigues en cliquant sur les [[liens]] internes.

## Organisation

| Dossier | Contenu |
|---|---|
| 00-Accueil | Point d'entree (carte du vault) |
| 01-Python | Notes sur le langage Python |
| 02-Algorithmie | Methodes de raisonnement, structures, complexite |
| 03-Exercices | Enonces, nos solutions commentees, fichiers .py |
| 04-Journal | Une note par session : ce qu'on a fait, ce qu'il reste |

## Methode de travail

Approche socratique : je pose des questions, tu construis la reponse.
Le but n'est pas que je code a ta place, mais que tu saches penser le code.

Commence par [[00-Accueil/Accueil|la note d'accueil]].
